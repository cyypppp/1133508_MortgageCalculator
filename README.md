# 1133508_MortgageCalculator

專案簡介
這是一個用 C# 寫的房貸計算機。它讓使用者只要輸入幾個數字，就能快速算出每個月要繳多少房貸，不用自己在那邊按計算機按到手酸。我特別設計了簡單的介面，而且有做防呆機制，就算使用者亂填數字，程式也不會直接崩潰。

核心功能

房屋總價輸入：支援整數與浮點數。
自備款計算：可依照需求選擇輸入自備款百分比或金額。
利率與期限：支援年利率百分比與貸款年限設定。
寬限期支持：可選填寬限期年數，計算前幾年只繳利息的狀況。
自動格式化：輸出結果包含千分位逗號，並精確至小數點後兩位。

計算結果判讀

貸款總金額：房屋總價減去自備款之剩餘金額。
每月應繳金額：依據本息平均攤還法計算之每月還款額。
首期本利：清楚顯示第一個月所需支付的利息與本金。
總支出統計：計算整個貸款期間的總利息支出與總還款金額。
